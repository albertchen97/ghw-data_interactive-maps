## Install Dependencies

```bash
pip install -r requirements.txt
```

---
*On Linux, if you encounter error messages regarding pycairo, try this:*

```bash
sudo apt install libcairo2-dev pkg-config python3-dev
```

## Build & Run

```bash
streamlit run main.py
```
